# Årsoppgave

Årsoppgave som er laget med node/express

## Production

### Install Dependencies
```bash
  npm install
```

### Start App
```bash
  npm run start
```

## Development

### Install Dependencies
```bash
  npm install
```

### Run Sass Compiler
```bash
  npm run sass
```

### Run Webpack
```bash
  npm run webpack
```

### Run Dev Server
```bash
  npm run serve
```
