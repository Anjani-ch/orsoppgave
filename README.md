# Årsoppgave

Årsoppgave som er laget med node/express

### Technologies Used:
- Node js
- Express
- EJS
- MongoDB
- Scss
- Webpack
- Passport js

## Production

### Install Dependencies
```bash
  npm install
```

### Start App
```bash
  npm run start
```

### Run Webpack
```bash
  npm run webpack-prod
```

## Development

### Install Dependencies
```bash
  npm install
```

### Run Sass Compiler
```bash
  npm run sass
```

### Run Webpack
```bash
  npm run webpack
```

### Run Dev Server
```bash
  npm run serve
```

### Run Webpack
```bash
  npm run webpack-dev
```
